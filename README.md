# github-training

`git checkout -b xxx/xxx`

`git switch xxx/xxx`

全てのファイルを登録
`git add .`
ステータスの確認
`git status`

1行のコミットメッセージと併せて記録する
`git commit -m 'xxx'`

`git push origin xxx/xxx`
ファイルの変更差分を確認する
`git diff`

コミット履歴を閲覧する
`git log`

コミットを取り消す
`git reset <commit id>`
ファイルの変更内容は保持される

ワーキングディレクトリの状態を直前のコミットまで戻す
`git reset --hard <commit id>`
作業内容は全て破棄される

作業途中でもブランチを切り替えられる
`git stash`
